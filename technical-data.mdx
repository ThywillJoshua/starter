## 9.2 Electrical specifications
### 9.2.1 Hand Washing Station
| | | |
|---|---|---|
| Input voltage [Vdc] | 12 | |
| Max power consumption [W] | 20 | |
| Standby mode consumption [W] | 1.5² | 6³⁴ |
| Average operating power1 [W] | 8.5² | 13³ |
| Operating temperature [°C] | 5 - 40 | |

¹ average of one hand wash with factory settings and assumed 8 sec. to come and leave.
² model without communication module and without automatic paper dispenser
³ model with communication module (touchscreen) and with automatic paper dispenser
