## 6.1 Maintenance lights

System status is displayed through maintenance lights using a certain color code. This code is described below.

<Frame>
  <img
    src="./images/Fig. 16 Blue maintenance light.png"
    alt="Maintenance light indicator"
  />
</Frame>

**Fig. 16 Blue maintenance light**

<Callout>
  The Docking Station also has two lights oriented to the sink that reproduce
  the same code as maintenance lights.
</Callout>

<AccordionGroup>
<Accordion title="6.1.1 Blue light turns on">
Possible reasons:
- **Consumable is empty:** Check if there is enough soap in the bag (see chapter 7.3.3 Insert soap bag) or if there is paper (see chapter 7.3.2 Refilling paper)
- **The SMU is not detected:** Check if the SMU is properly placed on the Docking Station and if the Docking Station is properly closed
</Accordion>

<Accordion title="6.1.2 Blue light blinks slowly (once every second)">
**Consumable is low**

Be aware that consumables are low and will need to be refilled soon.

</Accordion>

<Accordion title="6.1.3 Blue light blinks quickly (3 blinks and 1 pause)">
**Cleaning mode is activated**

The Hand Washing Station is in cleaning mode. During the cleaning mode, you can activate only the water-air mixture by activating the cleaning mode button. The hand-washing cycle is disabled.

</Accordion>

<Accordion title="6.1.4 Red light turns on">
  Possible errors: - **Defective parts / Error in electronic control:**
  <Steps>
    <Step>Try to reboot the system by switching it off.</Step>
    <Step>Wait for 3 seconds and switch the system back on.</Step>
    <Step>
      If the red light is still on after rebooting, please contact your service
      partner.
    </Step>
  </Steps>
</Accordion>

<Accordion title="6.1.5 Red light blinks slowly (once every second)">
- **Consumables has not been refilled:** It has been a week or more since one of the consumable has been detected to be empty and not refilled. Refill paper (see chapter 7.3.2 Refilling paper) or soap (see chapter 7.3.3 Insert soap bag).
</Accordion>
</AccordionGroup>

## 6.2 Troubleshooting

<AccordionGroup>
  <Accordion title="No function">
    - Check if the system is plugged to power. - Check if the system is switched
    on. - Check if maintenance lights are active. - Check if the white light
    above the sink is on when you stand in front of the system. - Check if the
    body sensor window is stained - Check if the docking station window is
    stained - Switch off the system, wait for 5 seconds and switch the system on
    again.
  </Accordion>
  <Accordion title="No / insufficient amount of water">
    - Check if water supply is turned on. - Check if there is leakage. - Check
    if the system parameters are correct.
  </Accordion>
  <Accordion title="No / insufficient soap emulsion">
    - Check whether there is enough soap in the soap bag. - Check if the soap
    bag was installed correctly in the docking station. - Check if the soap bag
    is already open for more than 6 months. - Check if the system parameters are
    correct. - Set the soap quantity to the maximum 5 ml (see 5.1 Factory
    settings) and launch 3 hand washing cycles, and then set back the system
    parameters.
  </Accordion>
  <Accordion title="Minor leakage in Docking Station">
    Check if there is water inside the docking station. Cut water supply
    immediately, switch off the system and contact your service partner or
    Smixin SA at once.
  </Accordion>
  <Accordion title="Leakage">
    Cut water supply immediately, switch off the system and contact your service
    partner or Smixin SA at once.
  </Accordion>
  <Accordion title="Soap leakage">
    <Steps>
      <Step>
        Switch off the system immediately and remove the soap bag, the docking
        station and the blue cover.
      </Step>
      <Step>Clean all leaked soap.</Step>
      <Step>
        Add a new soap bag and strongly screw the SMU (see chapter 7.3.3 Insert
        soap bag).
      </Step>
      <Step>Put back the docking station and blue cover.</Step>
    </Steps>
  </Accordion>
</AccordionGroup>

For further support, contact your service partner or Smixin SA at once, or go to the Smixin homepage.
