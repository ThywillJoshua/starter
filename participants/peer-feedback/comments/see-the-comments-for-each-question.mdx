[Source](https://www.start-pulse.com/en/resources/help/peer-feedback/comments/how-can-i-see-the-comments-for-each-question)

A rating must be given for each question. Participants can voluntarily add a comment while answering the question to further refine the rating.

<Callout variant="info" title="Who sees comments">
  Only feedback seekers can view comments in Peer Feedback.
</Callout>

<Steps>
  <Step title="Open Results">
    Go to the **Results** page and click the question you’re interested in.
  </Step>
  <Step title="Scroll comments">
    Scroll to read all comments for that question.
  </Step>
  <Step title="Navigate questions">
    Use the arrow in the upper-right to move to the next question’s comments.
  </Step>
</Steps>
