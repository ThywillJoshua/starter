[Source](https://www.start-pulse.com/en/resources/help/sicherheit-datenschutz1/security/is-my-data-protected-against-loss)

With a double server infrastructure, a reserve system and backups, Pulse has taken the relevant protective measures for your data and the operation of your Pulse license. All backups are encrypted according to the latest security standards and stored on several physically separated and redundantly mirrored data storage devices.
