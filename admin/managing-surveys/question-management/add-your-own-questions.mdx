<Callout variant="info" title="When to add custom questions">
  Use custom items to capture org-specific themes while keeping the core Pulse questions intact.
</Callout>

<Steps>
  <Step title="Open Questions">
    Go to **Admin** → **Questions**.
  </Step>
  <Step title="Create a question">
    Click **➕** below the search bar to add a new question.
  </Step>
  <Step title="Assign a topic">
    Pick the topic that best groups this question for reporting.
  </Step>
  <Step title="Add text and translations">
    Enter the question in the base language and add translations as needed.
  </Step>
  <Step title="Save and reuse">
    Save to make it available in the library for future surveys.
  </Step>
</Steps>

<Callout variant="warning" title="Missing translations">
  If a translation is missing, the question appears in the base language for those users.
</Callout>
