<Callout variant="info" title="Keep topics lean">
  Use a concise set of topics to keep navigation simple and reporting comparable.
</Callout>

<Steps>
  <Step title="Open Topics">
    Admin → **Topics** in the left menu.
  </Step>
  <Step title="Add a topic">
    Click **➕** under the search bar, name the topic, and add translations.
  </Step>
  <Step title="Save and apply">
    Save the topic, then use it when creating or editing questions.
  </Step>
</Steps>

<Callout variant="warning" title="Governance">
  Periodically merge or retire duplicate topics so the library stays clean.
</Callout>
