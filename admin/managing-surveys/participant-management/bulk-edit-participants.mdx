When you want to bulk edit the Manager, Team or Language of participants follow these steps.

**Bulk editing participants**

1. Tap your name on your top right corner and go to **Admin**
2. In the participants table, select the users you would like to edit by marking the boxes
3. Click on the **pencil icon** ( ✏️) to edit the users
4. Now you can choose which fields you want to modify and apply the changes to the selected users
