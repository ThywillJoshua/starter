As an admin, you can easily edit participant information directly in the tool.

<Callout variant="info" title="Use this for single edits">
  Perfect for correcting a name, team, or manager without re-importing the list.
</Callout>

<Steps>
  <Step title="Open Participants">
    Click your name (top-right) → **Admin** → **Participants**.
  </Step>
  <Step title="Edit an individual">
    Hover a row and click the **pencil** icon, or select the checkbox and use the top **pencil** icon.
  </Step>
  <Step title="Save changes">
    Update fields (name, team, manager, language) and click **Save**.
  </Step>
</Steps>

<Callout variant="info" title="Bulk edits">
  To update multiple people at once, select several rows and use the same **pencil** icon (see <a href="/admin/managing-surveys/participant-management/bulk-edit-participants">bulk edit</a>).
</Callout>
